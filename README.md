# PHYSICS SIMULATOR
A web development project. Built to understand projectile motion in a more visual and interative manner. Users can change the initial velocity and launch angke to observe how the projectile's path changes.

# FEATURES
- Adjustable velocity slider
- Launch Angle Slider
- Real-time projectile animation
- Interative controls

# INSPIRATION
11th standard physics book.

# BUILT WITH
HTML, JavaScript, and Canvas
